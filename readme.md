# XState Demo

## Dev
### Start server
```sh
node server.js
```

### Start App (the second terminal)
```
cd app

yarn

yarn serve
```

open http://localhost:8080


## Made with

* [xstate](https://github.com/davidkpiano/xstate)
* [@xstate/vue](https://github.com/davidkpiano/xstate/tree/master/packages/xstate-vue)
* [restify](http://restify.com/)
